# Nospree Legal

This repository contains the public legal pages for **Nospree**:

* Privacy Policy
* Terms of Use
* Account and data deletion instructions

These pages are hosted with GitHub Pages and used for app store review, user transparency, and support.

## Pages

* `index.html` — Legal pages index
* `privacy.html` — Privacy Policy
* `terms.html` — Terms of Use
* `delete-account.html` — Account and data deletion instructions
* `style.css` — Shared page styles
